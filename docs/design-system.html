RULES

DO
  • Use only the 5-shade palette
  • Keep all corners sharp (border-radius: 0)
  • GameFreak font always UPPERCASE
  • Pixel-style Unicode icons only
  • 3px borders for primary elements
  • Fira Code for all terminal/code
  • Scanline overlay on page background
DON'T
  • No color emoji anywhere on site
  • No rounded corners
  • No gradients
  • No shadows or drop-shadows
  • No lowercase in GameFreak font
  • No colors outside the palette
  • No generic fonts (Inter, Arial, etc.)
