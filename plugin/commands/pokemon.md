---
name: pokemon
description: Show your Pokemon companion's status, stats, and evolution progress
---

Read the file at ~/.statusmon/trainer.json to get the user's current Pokemon companion data.
Then describe their companion Pokemon in a fun, engaging way including:

1. The Pokemon's name, level, and XP progress
2. Its type(s) and a brief description
3. How close it is to evolving (or if it's in its final form)
4. A fun flavor text or encouragement about their journey

Keep it concise and Pokemon-themed. Use emoji for type indicators.
